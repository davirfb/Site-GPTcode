Um Framework para Análise de Segurança em Aplicações Web

Autores: Silva, J., Costa, M.

Resumo (Abstract)

Este artigo apresenta um novo framework para a automação da análise de vulnerabilidades em aplicações web modernas. Com a crescente complexidade das tecnologias de front-end e back-end, métodos tradicionais de varredura se mostram insuficientes. Nosso framework, denominado WebGuard, utiliza uma combinação de análise estática de código (SAST) e análise dinâmica de aplicações (DAST) para identificar falhas de segurança comuns, como SQL Injection, Cross-Site Scripting (XSS) e Insecure Direct Object References (IDOR), com uma taxa de detecção significativamente superior às ferramentas existentes.
